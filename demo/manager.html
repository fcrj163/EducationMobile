What Is TSS Mobile Application ?

'TSS Mobile Application' is a lite version of 'TSS Education System' for mobile device (Iphone or Android ). It aims to help Employees & Managers to get web system information via mobile device in time.

· Enhanced User Experience

· Direct Report From DB2

· Enhanced Interaction (System To User, User To User)
